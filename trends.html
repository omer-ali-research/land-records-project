Mortgages by type (FHA / VA) and period

Only counties with non-zero total rows are included. For each period, FHA and VA counts use date bucketing into period buckets. Type comes from the type column (or similar), else filename/link text. VA is also inferred from lender or notes (e.g. Administrator of Veterans Affairs) when type is wrong. FHA is checked before VA in text so "FHA" is not misread as VA. Rows with no FHA/VA label or no parseable year are omitted from these cells.
