Reported vs collected (FHA)

Counties are the same set as on the FHA/VA page (from trends.json). Reported values come from the Target counties tracker columns used on the Reported page. Collected is the FHA-only count for that period from digitized records (same bucketing as the FHA/VA type table). Within each period column, position on the horizontal axis is proportional to count; scale is shared across all visible counties in that column. Column subtitles (METRO, SMA, COUNTY) match the geography type of the tracker field for that period.
